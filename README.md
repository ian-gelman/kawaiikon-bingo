# Kawaii Kon Bingo Generator!
Does what it says on the tin.

## Setup
### Create virtual environment
```
python3 -m venv env
```
### Install required modules
```
pip install -r requirements.txt
```
### Add bingo squares
Make a file called `lines.txt` at the root of this repo and add each thing you want the generator to randomly pick from on its own line
### Run it
```
python3 bingus.py
```
Look for the finished card in the `output/` folder!
